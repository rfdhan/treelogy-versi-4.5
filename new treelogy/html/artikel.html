Mengapa Harus Install JDK?

Kalo kamu ngoding pake bahasa pemrograman Java di laptop kamu sendiri, berarti laptop kamu harus bisa melakukan compile koding Java tadi menjadi sesuatu yang dibisa dijalankan oleh laptop kamu. Kalo gak dicompile, gimana koding itu mau jalan? Nonsense! Supaya laptop kamu bisa compile, laptop kamu harus diinstall tools untuk compile. Tools itu ada di JDK. Jadi, kamu harus install JDK juga, kan? Jadi JDK cuma dipake untuk meng-compile dan menjalankan koding Java yang kita buat. Bukan untuk menulis koding Java, bukan!
